el **external_id** (resumen) y el **numero de ticket**.

 - **Método:** POST

 - **TOKEN:** HOHYgwd4atPalTxqi2IIspythpLAU7eP239oByfjk9VuveiOpfUTm4Yg0Szt  

Cuando haya culminado este procedimiento, el resumen de anulación con la boletas que lo conforman han sido aceptadas exitosamente por Sunat, 
es decir el (los) documento(s) han sido anulados.
